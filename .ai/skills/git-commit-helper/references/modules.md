# Project Modules

Use these as the `<scope>` in commit messages:

- **auth**: Authentication, OAuth2, and security configuration
- **profile**: User profile management, services, and controllers
- **db**: Database migrations, Liquibase, and JPA repository setup
- **model**: DTOs, entities, and adapters
- **config**: Global application configuration and exception handling
- **test**: Test infrastructure, base classes, and common test utilities
- **build**: Gradle build scripts, dependency management, and CI/CD
- **docs**: Markdown documentation and Javadoc
